# Fonts Directory

This directory contains web fonts used by the Novora Solutions website.

## Font Files

The following font files should be placed in this directory:

- `main-font.woff2` - Primary font in WOFF2 format
- `main-font.woff` - Primary font in WOFF format (fallback)
- `secondary-font.woff2` - Secondary font in WOFF2 format
- `secondary-font.woff` - Secondary font in WOFF format (fallback)

## Usage

These fonts are preloaded in the HTML head and referenced in the CSS.

## Font Licensing

Please ensure all fonts used have proper licensing for web use.
